# AI Resume Generator

AI Resume Generator is a simple web-based application that helps users generate professional resume summaries instantly based on their skills, experience, and job role.

This project is built using only frontend technologies and is ideal for students and freshers.

---

## Features
- Generate professional resume summaries
- Multiple resume tones
- Voice-enabled output using browser Text-to-Speech
- Clean and user-friendly UI
- No backend or API required

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter details and generate resume
